# basic-hack-tools

A small collection of quick scripts and "hacks" intended as small utilities or examples.

Contents
- hack.sh — tiny bash helper
- LICENSE — MIT
- .gitignore

Quick start
1. Download or clone the repo
2. Make scripts executable: `chmod +x hack.sh`
3. Run: `./hack.sh --help`

Examples
- ./hack.sh --say "Hello"
- ./hack.sh --timestamp
